This page documents every release of Tamperline.us. We ship updates regularly to improve your brewery management experience. Active subscribers are notified of major releases by email.

v1.4

February 2026

🚀 Latest release. This is the current version of Tamperline.us.

New Features

  • Demo video modal on the home page — watch the platform walkthrough without leaving the site
  • Batch duplicate — clone an existing batch record to quickly start a new brew of the same recipe
  • Booking confirmation emails now include a calendar (.ics) attachment for easy scheduling
  • Low-stock threshold alerts via email and in-dashboard notifications

Improvements

  • Pricing page redesigned with clearer feature comparison between Free and Pro plans
  • Faster inventory search — results now appear as you type with no page reload
  • Mobile dashboard layout improved for screens under 768px
  • Booking calendar now shows availability in the brewery's local timezone

Bug Fixes

  • Fixed an issue where batch logs with special characters in the recipe name failed to save
  • Resolved a display bug on the reports page when date ranges spanned multiple months
  • Fixed email reminder timing — reminders now send at the correct local time, not UTC
v1.3

January 2026

New Features

  • Sales analytics dashboard — track revenue by product, category, and time period
  • Multi-staff accounts — invite team members with role-based access (Admin, Manager, Staff)
  • CSV export for inventory and batch records
  • Recurring booking support for weekly or monthly taproom reservations

Improvements

  • Dashboard load time reduced by 40% through server-side caching improvements
  • Inventory table now supports sorting by any column
  • Improved onboarding flow with step-by-step setup guide for new accounts

Bug Fixes

  • Fixed a rare issue where batch fermentation timer reset unexpectedly on page refresh
  • Resolved stripe webhook failure for certain payment retry scenarios
  • Fixed footer navigation links on legal pages pointing to incorrect anchors
v1.2

December 2025

New Features

  • Equipment rental management — track kegs, taps, fermenters, and other rentable assets
  • Booking deposit collection via Stripe — require a deposit at time of reservation
  • Dark mode support across all dashboard screens
  • Print-friendly batch record sheets — generate a clean PDF from any batch log

Improvements

  • Calendar view added to the bookings section for a visual overview of upcoming reservations
  • Ingredient unit conversions (lbs/kg, gallons/litres) now configurable per account
  • Notification preferences page — choose which alerts you receive and how

Bug Fixes

  • Fixed inventory quantity not updating correctly after a batch is marked as complete
  • Resolved session timeout happening earlier than the configured 24-hour window
v1.1

November 2025

New Features

  • Batch log notes — add timestamped fermentation notes and gravity readings to any batch
  • Booking waitlist — automatically notify customers when a previously full slot opens up
  • Inventory reorder point tracking — set minimum stock levels per ingredient

Improvements

  • Booking confirmation page redesigned with clearer summary and next steps
  • Improved form validation across all data entry screens with inline error messages
  • API response times improved by an average of 120ms through query optimisation

Bug Fixes

  • Fixed an issue where deleting a booking did not free up the time slot in the calendar
  • Resolved incorrect total calculation in the weekly inventory summary report
  • Fixed mobile keyboard pushing the login form off-screen on iOS Safari
v1.0

October 2025 — Initial Launch

🍺 The first public release of Tamperline.us — built specifically for craft breweries, taprooms, and small retailers.

Core Platform Features

  • Ingredient inventory tracking — hops, malt, yeast, adjuncts, packaging materials
  • Batch production logging — recipes, brew dates, fermentation targets, and status
  • Taproom booking management — time slots, capacity limits, customer details
  • Stock level dashboard with at-a-glance inventory health indicators
  • Free plan with up to 10 bookings/month; Pro plan with unlimited access
  • Secure authentication with email and password; account recovery via email
  • Basic sales and inventory reports exportable to CSV
  • Mobile-responsive design for use on tablets and phones in the brewery
